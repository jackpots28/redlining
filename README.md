## Run Locally

Clone the project

```bash
  git clone https://github.com/jackpots28/redlining.git
```

Go to the project directory

```bash
  cd redlining
```

Install dependencies

```bash
  python3 -m pip install -r requirements.txt
```

Start the flask server

```bash
  python3 web_app.py
```
